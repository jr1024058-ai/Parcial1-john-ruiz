# Parcial1-john-ruiz

Un proyecto simple de página web estática que muestra una lista de servicios con posibilidad de agregarlos a una lista y validar un formulario de contacto con JavaScript

## Estructura del examen

- `index.html` — Página principal con contenido y formulario
- `css/styles.css` — Estilos básicos de la página
- `js/script.js` — Lógica de interacción (modo oscuro, lista de servicios, validación de formulario)
- `assets/` — Imágenes de productos

## Una pequeña guia

1. Abre `index.html` en tu navegador.
2. Navega por los productos y prueba las interacciones
   - Modo oscuro
   - Enviar el formulario de contacto

## Funcionalidades (lo que se pudo)

- Modo oscuro aplicando estilos al `body`
- Contador de servicios escogidos agregados a la lista de servicios
- Validación básica de formulario con alertas

## cositas

- El proyecto está pensado como un ejercicio evaluativo
- La validación del formulario es totalmente local
